# Learning Loops

Use this reference when the user wants a plan, repeated practice, material reuse, or progress tracking.

## Core Principle

English improves fastest when input becomes output, output receives feedback, and feedback becomes the next attempt.

The default lesson loop:

1. Goal.
2. Input.
3. Output.
4. Feedback.
5. Retry.
6. Record.

## Session Design

Keep each session narrow:

1. One target.
2. One small input.
3. One visible output.
4. One main correction.
5. One retry task.

Good output tasks include:

1. Say it in English.
2. Write a short answer.
3. Summarize the paragraph.
4. Retell from memory.
5. Fill a gap.
6. Correct a sentence.
7. Choose the best expression and explain why.

## Material Reuse

One good article, transcript, email, essay, or dialogue can support several rounds:

1. Reading comprehension.
2. Vocabulary and collocation extraction.
3. Dictation or transcript reconstruction.
4. Spoken retelling.
5. Writing imitation.
6. Quiz and review.

Reuse the same material when the learner needs depth, not novelty.

## Weekly Loop

A simple weekly cycle:

1. Speaking plus corrected repetition.
2. Close reading plus useful expressions.
3. Scenario simulation with the same topic.
4. Quiz or flashcards.
5. Short writing and revision.
6. Review recurring mistakes.

## Learning Record

Record:

1. Material used.
2. Output produced.
3. Main weak point.
4. Useful expression.
5. Retry result.
6. Next review task.
